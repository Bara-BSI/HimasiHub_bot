# HimasiHub_bot
## Requirements
1. node.js
2. npm
## How To
1. Clone dan buka
```bash
git clone https://github.com/Bara-BSI/HimasiHub_bot.git
cd HimasiHub_bot
```
2. Copy file .env
```bash
cp .env.example .env
```
3. Edit file .env, isikan token bot telegram
4. Install modul npm
```bash
npm install
```
6. Jalankan script bot
```bash
node bot.js
```
